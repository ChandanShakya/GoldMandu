# GoldMandu Project

- Idea on : **2022-05-20**

## Requirements

1. View today's gold price:
    - Calculated from gold stock and converted to Nrs
    - Scraped from Kathmandu Gold and Silver Association
    - Must include time of updating
2. Page with the converter for:
    - Currency conversion
    - Unit Conversion
3. Static and hosted
4. Responsive
5. Easy to use UI
